ThinkPHP 8 + MySQL

Starter kit untuk belajar ThinkPHP

Contoh CRUD artikel lengkap dengan model, validasi, view, routing RESTful.

Buat artikel pertama Lihat daftar artikel

Cara

  1. Salin konfigurasi env 
    cp .example.env .env
  2. Update kredensial MySQL di .env dan jalankan SQL di database/schema.sql
  3. Jalankan server dev 
    php think run
  4. Buka http://localhost:8000 dan coba menu Artikel

Checklist belajar ThinkPHP

Ikuti urutan ini supaya konsep-konsep inti terasa utuh.

Routing

Resource dan endpoint

route/app.php untuk resource articles dan rute utama.

Controller

CRUD + validasi

app/controller/Article.php memuat alur CRUD, filter, dan flash message.

Model & ORM

Schema, scope, timestamp

app/model/Article.php mengatur schema, scope, dan timestamp otomatis.

Validasi

Rule yang tegas

app/validate/ArticleValidate.php menjaga input bersih dan konsisten.

View

Template dan layout

view/articles menampilkan form, daftar, dan detail dengan layout.
